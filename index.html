Personal Projects

Mood

A mood journal that stores journal posts into a calendar and tracks emotions on a daily basis. Its purpose is to help people be more self-aware of their mood and hopefully brings positive realizations in their lives. I built this app when I was coming out of depression. I was journaling my eventful activites and it helped me bring positive changes into my life.

    Front end:
  • HTML / Javascript / jQuery
  • CSS / Responsive / AJAX
    Back end:
  • Node.js / Express / Mongo / Mongoose / Passport.js / JWT / RESTful Api / bcrypt.js
  • Mocha / Chai

EDM Finder

This app finds EDM artists playing in certain cities and is able to play their music using Youtube's API. EDM Finder helps less navigation and noise on artists that aren't known to the user. I created this app because Spotify and Soundcloud did not have a big database of EDM artists unlike Youtube. Youtube's advantage over the other two is it displays music videos.

    Front end:
  • HTML / Javascript / jQuery
  • CSS / Responsive / AJAX
